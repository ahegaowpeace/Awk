## data2
同じレコードが存在する行と、その行番号を出力
```
cat data2
0.59320025 0.67320025
0.59328825 0.67200025
0.59118825 0.67203325
0.77328825 0.67207125
0.77328825 0.67207125
0.59320025 0.67320025
0.59320025 0.67320025
0.59320025 0.67320025
```
```
$ cat data2 | awk 'a[$0]{print a[$0],NR,$0}{a[$0]=NR}'
4 5 0.77328825 0.67207125
```
